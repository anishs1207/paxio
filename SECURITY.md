# Security Policy

## Supported Versions

Only the latest version of Paxio is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of Paxio seriously. If you believe you have found a security vulnerability, please report it to us as described below.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send an email to [anishs1207@gmail.com](mailto:anishs1207@gmail.com). 

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Any potential impact if exploited.

We will acknowledge receipt of your report within 48 hours and provide a timeline for resolution. Once the vulnerability has been resolved, we will credit you for the discovery (unless you prefer to remain anonymous).

Thank you for helping keep Paxio secure!
